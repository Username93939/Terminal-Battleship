# Terminal Battleship
This is a terminal version of the board game, complete with a fairly competent bot

To run it, either download the pre-compiled executables or compile it with

```g++ *.cpp -o "Battleship"```

then run

```./Battleship```

Note the pre-compiled executables may not be the same version as the code here, as I am not re uploading a new mac version for every tiny improvement because it is rather annoying to compile code for macs without having a mac (which I do not have)
